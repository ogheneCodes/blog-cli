# Blog CLI Application

## Description
A simple Command Line Interface (CLI) blog application built with Node.js.

## Features
- Create blog posts
- View blog posts
- Update blog posts
- Delete blog posts
- Data stored in JSON file

## Technologies
- Node.js
- Inquirer
- File System (fs)

## How to Run
1. Install dependencies:
   npm install

2. Run app:
   node blog.js
